# data-journalism
知达教育 - 数据新闻实践

## 我们是谁？

知达教育是一家从事数据人才培养以及企业数字化转型咨询服务的机构。

知达，意为明智通达、知识达人，同时也是星球大战中绝地武士 Jedi 的谐音。我们致力于成为数据界的绝地学院，培养应用型数据人才，用数据驱动个人与企业的发展。


## 为什么要开源？

数据新闻引起大家的关注是从2013年逐步开始的，到2015年进入上升期（参考百度指数）。相比国外的成熟发展，数据新闻在国内只能说刚刚开始。

最近几年，相关的技术和环境原来越成熟，我们有了更多的数据获取渠道，有了容易上手的可视化工具 Tableau，有了灵活强大的设计软件 Sketch，同时也有了更多的媒体专栏渠道（公众号、专栏、博客）。但实际去做时，可以参考学习的资料却很欠缺。

在知达教育，我们的主营业务其实并不是数据新闻，但我们有经验丰富的数据可视化分析师，有文笔精湛的文字工作者，有擅长手绘的设计师，同时大家都喜欢观察思考，希望通过数据新闻来传播数据思维的理念。实践是最好的学习，所以我们决定不仅从公司的专栏文章开始写起，也把我们在这个过程中的想法、数据、分析报表、设计文件、经验心得，以开源项目的方式分享给大家。

希望你会喜欢。
